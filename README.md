This project was developed using Prompt Engineering (Vibe Engineering).

Instead of generating the entire website using a single prompt, each section was developed independently using carefully engineered prompts.

The prompts were refined multiple times to improve:

- Layout
- Responsiveness
- User Experience
- JavaScript Functionality
- CSS Styling
- Code Organization
